# ROBO-ONE Beginners Auto BLE PC Controller

## Introducción

Este proyecto es un **software controlador de BLE para PC** diseñado para estudiantes que quieren experimentar con **robots autónomos de ROBO-ONE Beginners**.

---

## Sobre este proyecto

Actualmente estoy desarrollando un robot para competir en categorías autónomas como **ROBO-ONE Beginners**.

Durante el desarrollo encontré varios problemas relacionados con **BLE (Bluetooth Low Energy)**.

Especialmente:

* Las aplicaciones BLE de smartphone son incómodas para controlar robots
* Las pruebas y depuración de comunicación son difíciles
* La integración con programas de desarrollo es limitada

Por esta razón desarrollé un **controlador simple en Python que permite manejar la comunicación BLE desde un PC**.

---

## Propósito del repositorio

Este repositorio publica el **software controlador BLE del lado del PC**.

Sin embargo, este software **todavía está en desarrollo** y puede ser ligeramente diferente de la versión que uso actualmente.

La versión publicada aquí se mantiene intencionalmente:

* simple
* fácil de entender
* adecuada como punto de partida para modificaciones

---

## Filosofía del proyecto

No considero este software como una herramienta completa.

La idea es que funcione como **un punto de partida** para personas que quieran:

* experimentar con comunicación BLE
* desarrollar robots autónomos para ROBO-ONE
* aprender sobre control de robots

---

## Descripción general

Este software permite **controlar un robot mediante BLE desde un PC**.

Aunque normalmente la comunicación BLE se realiza mediante aplicaciones móviles, usar un PC permite:

* mayor flexibilidad
* mejor depuración
* integración con programas de control autónomo

---

## Funciones principales

* Conexión BLE desde PC
* Envío de comandos al robot
* Entorno para experimentación continua
* Base para desarrollo de control autónomo

---

## Requisitos

* Python 3
* PC con soporte BLE
* Robot ROBO-ONE Beginner con BLE
